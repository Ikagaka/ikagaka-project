# ikagaka-project

イカガカ関連コンポーネントをまとめて開発するLernaプロジェクト

## 必要

- node.js >= 8
- git

## インストール

最初にツールをインストール。
リポジトリのクローン（`npm run prepare`）も同時に行われます。

```bash
npm install
```

次に依存関係をインストールします。（3GBくらいになる）

```bash
npm run bootstrap
```
