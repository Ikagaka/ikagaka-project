# ikagaka-project

イカガカ関連コンポーネントをまとめて開発するLernaプロジェクト

## Install

最初にツールをインストール

```bash
npm install
```

まずリポジトリをクローンしてくる

```bash
npm run prepare
```

次に依存関係をインストールする（3GBくらいになる）

```bash
npm run bootstrap
```
